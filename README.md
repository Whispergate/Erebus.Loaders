# Erebus.Loaders
Loaders developed for the Erebus Wrapper Project.

## Specifications
### Erebus.ClickOnce

- Fetches remote payload for execution
- Executes DLL/Shellcode in memory
- Triggers file to run

### Erebus.Shellcode

- Can fetch remote payload
- Executes shellcode in memory
- Self Removal after execution is complete (TBD)
